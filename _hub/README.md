# _hub

Central de inteligência do e-commerce. Dashboards hospedados via GitHub Pages, sincronizados com esta pasta.

## Estrutura

| Pasta | Conteúdo |
|---|---|
| `dashboards/` | Dashboards HTML — linkáveis e atualizados em tempo real via GitHub Pages |
| `relatorios/` | Análises finais em markdown e PDFs gerados pelos agentes |
| `dados/` | Planilhas, dados consolidados e histórico para consulta |
| `kpis/` | Metas e acompanhamento por marca, canal e período |

> Dados históricos de KPIs ficam em `dados/` para facilitar consulta cruzada.
